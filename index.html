🚀 Guide de déploiement complet

Suivez ces étapes dans l'ordre pour mettre la plateforme en ligne aujourd'hui.

  1. Google OAuth : console.cloud.google.com → Create Project → APIs & Services → Credentials → Create OAuth 2.0 Client ID → Web Application. Authorized redirect URI : https://VOTRE-ID.supabase.co/auth/v1/callback
  2. Supabase Google Provider : Dashboard → Authentication → Providers → Google → coller Client ID & Client Secret → Save.
  3. SQL : Supabase → SQL Editor → New Query → copier le script ci-dessous → Run.
  4. GitHub : Créez un repo privé → uploadez index.html + netlify.toml (tous deux dans ce zip).
  5. Netlify : netlify.com → Add new site → Import from Git → sélectionnez votre repo → Deploy.
  6. URL callback : Supabase → Authentication → URL Configuration → Site URL = votre URL Netlify → Redirect URLs = même URL.
  7. ✅ Partagez l'URL Netlify à votre collègue. Il se connecte avec son compte Google, c'est tout.

Script SQL

-- RegistreDPO · Tables (Supabase SQL Editor) CREATE TABLE IF NOT EXISTS traitements ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, created_at TIMESTAMPTZ DEFAULT now(), created_by UUID REFERENCES auth.users(id), updated_at TIMESTAMPTZ, updated_by UUID REFERENCES auth.users(id), nom TEXT NOT NULL, responsable TEXT NOT NULL, finalite TEXT NOT NULL, description TEXT, date_creation DATE, base_legale TEXT, donnees_sensibles BOOLEAN DEFAULT false, categories TEXT[], personnes TEXT, duree_conservation TEXT, aipd_requise TEXT DEFAULT 'Non', transfert_hors_ue BOOLEAN DEFAULT false, sous_traitants TEXT, mesures_securite TEXT, statut TEXT DEFAULT 'Actif' ); CREATE TABLE IF NOT EXISTS audit_log ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, created_at TIMESTAMPTZ DEFAULT now(), user_email TEXT, user_name TEXT, action TEXT, traitement_nom TEXT, details TEXT ); ALTER TABLE traitements ENABLE ROW LEVEL SECURITY; ALTER TABLE audit_log ENABLE ROW LEVEL SECURITY; CREATE POLICY "authenticated_users" ON traitements FOR ALL USING (auth.uid() IS NOT NULL); CREATE POLICY "authenticated_users" ON audit_log FOR ALL USING (auth.uid() IS NOT NULL);

netlify.toml

# netlify.toml — root of your GitHub repo [build] publish = "." [[redirects]] from = "/*" to = "/index.html" status = 200 [[headers]] for = "/*" [headers.values] X-Frame-Options = "DENY" X-Content-Type-Options = "nosniff" Referrer-Policy = "strict-origin-when-cross-origin"
